Mancala

An interactive mancala game for your desktop created as part of a course project to demonstrate the use of jQuery. This is meant to be a 2-player game, where marbles are moved around the virtual game board with a single click.

pitched
Github repo > | Live site >

pitched is a campsite review app that was built for a course project to practice using EJS and MVC architecture. Users can register with the site and post their own pictures and thoughts on camping spots all over the world. Members of the site also have the ability to delete and edit their own reviews and comment on others.

Express, Bootstrap, EJS, Node.js, MongoDB, Mongoose

Wedding website
Github repo > | Live site >

This website was built for my two friends and their upcoming wedding. To help them with the wish for a paperless wedding, guests can send their rsvps through the site as long as they provide a valid email to associate with the invite. It's also fully responsive on mobile and tablet devices.

AngularJS, Ruby on Rails, PostgreSQL
